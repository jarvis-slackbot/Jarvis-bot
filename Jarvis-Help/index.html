Jarvis - Interactive Slack Bot


Project Overview:

The goal of this project is to develop the infrastructure and code supporting an interactive Slack bot for use by teams operating in a DevOps and cloud-oriented software development environment. This bot would be a valuable resource for developers, able to report on the health and development lifecycle of the teamâ€™s products, as well as trigger various related actions, through a simple, easy-to-use chat interface. Possible features include, but are not limited to:
1. Report on the status of a cloud-based environment: recent changes, instance status, automated health checks, etc.
2. Report on the status of a particular change: build time and result, test results, deployment status, etc.
